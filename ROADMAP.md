# Maher Hero — Project Baseline

آخر تحديث مرجعي: 14 يوليو 2026

## الحالة الحالية

- الإصدار المرجعي: Maher Hero v2.0.
- النشر التلقائي مفعّل عند كل Push إلى فرع `main` عبر GitHub Actions ثم Vercel Deploy Hook.
- السوق الأمريكي يعتمد على Alpaca.
- السوق السعودي يعتمد حاليًا على Yahoo Finance كمصدر غير رسمي وقد تكون البيانات متأخرة.
- محرك التقييم يعمل من الخادم، وليس من بيانات يرسلها المتصفح.

## ما تم تنفيذه

- إصلاح منطق المقاومة التالية بعد الاختراق.
- استبعاد شمعة الخمس دقائق غير المكتملة من حساب المؤشرات والحجم.
- استخدام Wilder RSI وتحسين شروط MACD.
- ربط السوق السعودي بالواجهة ومسار التحليل.
- نقل التقييم وجلب بيانات السوق إلى الخادم.
- إضافة Rate Limiting لحماية OpenAI ومسار التحليل.
- إضافة تصنيفات: شراء مشروط، مراقبة، انتظار إعادة اختبار، متأخر، استبعاد.
- إضافة تحذيرات المطاردة وRVOL وتفاصيل توزيع الدرجة.
- تحسين الأهداف وربطها بالمقاومات وإدارة المخاطر.
- إصلاح مسار `/api/push/test` ومنع فشل البناء بسبب العناوين العربية في HTTP Headers.
- تحويل إشعارات ntfy إلى JSON لدعم النص العربي بصورة آمنة.

## قواعد التشغيل

- لا ينفذ النظام أوامر شراء أو بيع تلقائيًا.
- أي توصية هي دخول مشروط ويجب التحقق من السعر والحجم في منصة التداول لحظة التنفيذ.
- لا يتم اعتماد فرصة قوية إلا عند توافق الاتجاه والزخم والحجم والاختراق والمسافة الفنية وإدارة المخاطر.
- بيانات السوق السعودي يجب التحقق منها قبل التنفيذ بسبب كون المصدر الحالي غير رسمي.

## المرحلة القادمة — Maher Hero v3.0

1. سجل صفقات وقاعدة بيانات للنتائج الفعلية.
2. اختبار تاريخي Backtesting قابل للقياس.
3. متابعة الصفقة بعد الدخول وتنبيهات نقل الوقف والبيع الجزئي.
4. توسيع فحص السوق الأمريكي مع فلترة السيولة وLow Float والأخبار.
5. تحسين تحليل الأطر الزمنية 5 دقائق و15 دقيقة وساعة ويومي.
6. لوحة أداء تعرض نسبة النجاح ومتوسط الربح والخسارة وProfit Factor.

هذا الملف هو المرجع الرسمي لأي تعديل جديد على المشروع.

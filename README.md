# Maher Hero AI Stock Scanner

نسخة أولية لموقع **استراتيجية ماهر هيرو** لتحليل فرص الأسهم وترتيبها من 100، ثم توزيع رأس المال وحساب الكمية ووقف الخسارة الفني.

## الموجود حاليًا

- واجهة عربية متجاوبة.
- اختيار السوق الأمريكي أو السعودي.
- إدخال رأس المال ونسبة المخاطرة.
- محرك تقييم ماهر هيرو محلي من 100.
- استبعاد الحركة المتأخرة وعدم مطاردة السهم.
- ترتيب أفضل 3 فرص.
- حساب تخصيص مبدئي وكمية مقترحة ووقف فني تقديري.
- ربط خادمي آمن مع OpenAI Responses API.
- بيانات تجريبية إلى حين ربط مزود بيانات سوق مباشر.

## التشغيل المحلي

```bash
npm install
cp .env.example .env.local
npm run dev
```

ثم افتح `http://localhost:3000`.

## إعداد OpenAI

أنشئ مفتاحًا جديدًا، ثم ضعه في ملف `.env.local` فقط:

```env
OPENAI_API_KEY=your_new_key
OPENAI_MODEL=gpt-5-mini
```

لا تضع المفتاح في ملفات الواجهة، ولا ترفعه إلى GitHub.

## المرحلة التالية

1. اختيار مزود بيانات لحظية للسوق الأمريكي.
2. بناء Market Scanner يجلب الأسهم النشطة تلقائيًا.
3. حساب RSI وMACD والحجم وATR والدعم والمقاومة من الشموع.
4. إعادة تقييم الفرص كل دقيقة أو عند تغير الزخم.
5. إضافة سجل صفقات وتنبيهات.
